# Microsoft Copilot Studio サンプル・テンプレート集

## 概要
Copilot Studio のサンプルやテンプレートなどを日本語で提供します。

## サポート

これらのサンプルの構築に使用されている基盤機能やコンポーネント（Copilot Studio ボット、Power Platform の製品や機能など）は完全にサポートされていますが、サンプル自体はこれらの機能の実装例を示すものです。お客様、パートナー、コミュニティの皆様は、これらの機能を利用およびカスタマイズして、自組織向けの機能を実装できます。

次のような問題が発生した場合:

- **サンプルの使用**: 直接イシューとしてこちらであげてください。（Microsoft サポートはサンプル自体に関する問題には対応しませんが、関連する基盤プラットフォームや機能の問題には対応します。）
- **Microsoft のコア機能**: 通常のチャネルを通じて Microsoft サポートにお問い合わせください: [Community help and support for Microsoft Copilot Studio](https://learn.microsoft.com/en-us/microsoft-copilot-studio/fundamentals-support)。

### セキュリティ問題の報告

**セキュリティ脆弱性は、公開 GitHub issue を通じて報告しないでください。**

代わりに、Microsoft Security Response Center（MSRC）へ [https://msrc.microsoft.com/create-report](https://msrc.microsoft.com/create-report) から報告してください。

ログインせずに提出したい場合は、[secure@microsoft.com](mailto:secure@microsoft.com) 宛てにメールを送信してください。可能であれば、当社の PGP キーでメッセージを暗号化してください。PGP キーは [Microsoft Security Response Center PGP Key page](https://www.microsoft.com/en-us/msrc/pgp-key-msrc) からダウンロードできます。

通常、24 時間以内に返信を受け取れるはずです。何らかの理由で返信がない場合は、元のメッセージが受信されていることを確認するため、メールで再度お問い合わせください。追加情報は [microsoft.com/msrc](https://www.microsoft.com/msrc) で確認できます。
